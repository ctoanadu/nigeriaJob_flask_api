BeeKin Job API Documentation

Introduction

The BeeKin Job API allows users to retrieve information about job postings on the BeeKin job board. The API provides endpoints for filtering, sorting, and pagination of job postings.

Authentication

The BeeKin Job API does not require authentication.

Base URL

All API endpoints are relative to the following base URL:

http://localhost:5000

Endpoints

Filter Jobs

Retrieves a list of job postings based on location and technology filters.

Endpoint HTTP Method Parameters Description
/jobs/filter GET location (string, optional), technology (string, optional) Retrieves a list of job postings based on the specified location and/or technology filters. If no filters are specified, returns all job postings.

Example Request

GET /jobs/filter?location=Lagos&technology=Python
